# 0003 — Shader 是技术分水岭，学习计划 Lv4 强化

确认 Shader 是 3D 前端的技术分水岭：
- 只会 API → 10k-18k（可替代性强）
- 能写 Shader → 15k-25k（有竞争力）
- Shader+性能+架构 → 25k-50k+（稀缺）

**Lv4 Shader 部分大幅扩展**：
- 从 5 个 Demo 扩展到 13 个专项（10 Shader + 3 性能）
- 新增核心参考：古柳 Shader 系列（入门+着色器基础）、The Book of Shaders、Shadertoy、Inigo Quilez
- 关键新增：attribute/uniform/varying 数据传递、纹理采样、噪声算法（值噪声/柏林噪声/Simplex/fBm）、6 种经典特效
- 总 Demo 数：35 → 38

**Implications**：
- Lv4 学习周期从 12-18 天延长到 15-20 天
- Shader 专项是学习计划中最重要的投入，决定了薪资天花板
- 噪声算法是 Shader 的核心，值得花最多时间
