# Mission: Three.js 3D Web 开发 — 从前端到3D工程师

## Why
作为一名前端开发者，希望系统掌握 Three.js 和 WebGL 3D 开发能力，从"会写页面"进阶到"能独立交付3D可视化商业项目"，在数字孪生、智慧城市、3D产品展示等热门赛道获得职业竞争力（BOSS直聘上3D前端岗位薪资 20k-50k+）。

## Success looks like
- 能独立从零搭建一个完整的3D商业项目（数字孪生园区 / 3D产品官网），并部署上线
- 能理解 WebGL 渲染管线底层原理，编写自定义 Shader 特效
- 能对 3D 场景进行性能优化（实例化渲染、LOD、内存管理），移动端流畅运行
- 具备 BOSS直聘上 3D 前端岗位要求的核心技能栈，简历有竞争力
- 在 GitHub 上拥有至少一个可展示的 3D 开源项目

## Constraints
- 已有前端基础（HTML/CSS/JS/TS），无需从零学前端
- 每天可投入 2-3 小时学习
- 学习周期：预计 3-4 个月完成全部 5 个等级
- 偏好动手实操，不喜欢纯理论

## Out of scope
- Unity/Unreal 等游戏引擎开发（聚焦 Web 端 3D）
- 深度学习/图形学论文级别的理论研究
- 后端开发（Go/Java/Node.js 服务端）
