## 核心改进

| 原方案 | 新方案 |
|--------|--------|
| 一上来就手写 WebGL 原生 | 先用 Three.js 10行代码出效果 |
| 理论 → 实践 | Demo → 理解原理 → 再深入 |
| 每个阶段 1个大项目 | 每个阶段 5-8个小 Demo + 1个综合项目 |

## Lv1 — Three.js 快速上手（3天，信心建立期）

> **目标**：会用 Three.js 搭场景，感受到"写几行代码就能出3D效果"的成就感
> **难度**：★☆☆☆☆

### 1.1 第一个3D场景：旋转的立方体（30分钟）
**Demo**: `demo-01-hello-cube.html`
- 引入 Three.js CDN
- 创建场景、相机、渲染器
- 添加一个立方体
- 让它旋转
- **收获**：10行代码看到3D效果，极大信心提升

### 1.2 认识几何体家族（30分钟）
**Demo**: `demo-02-geometries.html`
- 立方体、球体、圆柱体、圆锥体、圆环、平面
- 每个几何体的关键参数（长宽高、半径、分段数）
- 用不同颜色区分
- **收获**：知道 Three.js 能创建哪些基本形状

### 1.3 光影魔法（30分钟）
**Demo**: `demo-03-lights.html`
- 环境光（照亮所有面）
- 平行光（模拟太阳）
- 点光源（模拟灯泡）
- 聚光灯（模拟手电筒）
- 无光 vs 有光对比
- **收获**：理解光照对3D场景真实感的重要性

### 1.4 鼠标控制视角（30分钟）
**Demo**: `demo-04-orbit-controls.html`
- OrbitControls：拖拽旋转、滚轮缩放、右键平移
- 对比有无控制器的差异
- **收获**：让场景可交互

### 1.5 材质初探（45分钟）
**Demo**: `demo-05-materials.html`
- 基础材质（MeshBasicMaterial）
- 标准材质（MeshStandardMaterial）
- 线框材质（wireframe）
- 改变颜色、粗糙度、金属感
- **收获**：理解材质决定物体外观

### 1.6 🎯 综合项目：3D 产品展示台（1小时）
**Demo**: `project-01-product-showcase.html`
- 多个几何体排列
- 光照+阴影
- 鼠标旋转观察
- 点击切换颜色
- **收获**：第一个完整小项目，可展示

## Lv2 — Three.js 深入理解（5-7天）

> **目标**：理解 Three.js 的核心概念，能独立搭建复杂场景
> **难度**：★★☆☆☆

### 2.1 坐标系统与变换（45分钟）
**Demo**: `demo-06-transforms.html`
- position / rotation / scale
- 世界坐标 vs 局部坐标
- 父子关系（Group）
- 可视化坐标轴
- **收获**：精确控制物体位置

### 2.2 纹理贴图（45分钟）
**Demo**: `demo-07-textures.html`
- 颜色贴图（给立方体贴上木纹/金属/石头）
- 法线贴图（凹凸感）
- 环境贴图（反射）
- 贴图加载与跨域处理
- **收获**：让物体有真实质感

### 2.3 动画系统（45分钟）
**Demo**: `demo-08-animation.html`
- requestAnimationFrame 原理
- 物体动画（移动/旋转/缩放）
- 相机动画（漫游）
- 缓动函数
- **收获**：让场景动起来

### 2.4 粒子系统（45分钟）
**Demo**: `demo-09-particles.html`
- 创建粒子（Points）
- 粒子材质（SpriteMaterial）
- 下雨/下雪/星空效果
- **收获**：掌握粒子特效

### 2.5 文字与标注（30分钟）
**Demo**: `demo-10-text-labels.html`
- 3D 文字（TextGeometry）
- CSS2D 标签（始终面向屏幕）
- Sprite 精灵标注
- **收获**：3D场景中的信息展示

### 2.6 阴影完全指南（45分钟）
**Demo**: `demo-11-shadows.html`
- 阴影的完整配置流程
- 软阴影 vs 硬阴影
- 阴影性能优化
- **收获**：掌握阴影系统

### 2.7 🎯 综合项目：3D 房间漫游（2小时）
**Demo**: `project-02-room-tour.html`
- 用几何体搭建房间（地板+墙壁+家具）
- 光影+阴影
- 第一人称视角漫游（WASD行走）
- 粒子灰尘效果
- **收获**：完整的3D场景搭建能力

## Lv4 — WebGL 底层与性能优化（10-15天）

> **目标**：理解底层，能做性能优化，写自定义特效
> **难度**：★★★★☆

### 4.1 WebGL 渲染管线（1小时）
**Demo**: `demo-17-webgl-pipeline.html`
- 手写原生 WebGL 三角形
- 顶点着色器 vs 片元着色器
- GPU 如何工作
- **收获**：理解 Three.js 的底层

### 4.2 自定义着色器入门（1.5小时）
**Demo**: `demo-18-shader-basics.html`
- ShaderMaterial 基本用法
- uniform 变量传递
- 颜色渐变着色器
- 顶点扰动效果
- **收获**：开始写自己的 Shader

### 4.3 着色器特效（2小时）
**Demo**: `demo-19-shader-effects.html`
- 溶解效果
- 波纹效果
- 描边效果
- 霓虹发光
- **收获**：掌握常用 Shader 特效

### 4.4 后处理特效（1小时）
**Demo**: `demo-20-postprocessing.html`
- EffectComposer 管线
- 辉光（Bloom）
- 景深（DOF）
- 画面调色
- **收获**：提升画面品质

### 4.5 性能优化（1.5小时）
**Demo**: `demo-21-performance.html`
- 1000个立方体：普通 vs 实例化（InstancedMesh）
- FPS 监控
- 几何体合并
- 视锥剔除
- **收获**：让场景跑得流畅

### 4.6 贴图与资源优化（1小时）
**Demo**: `demo-22-texture-optimization.html`
- 贴图 2 的幂次方规范
- 纹理压缩
- 资源销毁（防止内存泄漏）
- SPA 路由切换时的资源清理
- **收获**：避免线上事故

### 4.7 物理引擎（1.5小时）
**Demo**: `demo-23-physics.html`
- Cannon-es 基础
- 重力+碰撞+弹跳
- 多米诺骨牌效果
- **收获**：让场景有物理真实感

### 4.8 🎯 综合项目：粒子特效秀（3小时）
**Demo**: `project-04-particle-show.html`
- 粒子系统+自定义Shader
- 辉光后处理
- 性能优化
- 鼠标交互
- **收获**：展示技术深度的作品

## ⚠️ 核心原则

1. **每个 Demo 都要亲手敲代码**，不要复制粘贴
2. **先跑通，再理解**，最后才优化
3. **遇到报错是好事**，排查过程学得最扎实
4. **每完成一个 Demo 就 commit 一次**，积累 GitHub 贡献
5. **不要跳级**，但同一级内可以自由调整顺序
6. **Lv3 完成后就可以投简历试水**，边学边面
