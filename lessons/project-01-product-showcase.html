📖 项目结构解析

这个项目的代码结构是你后续所有 Three.js 项目的基础模板：

// 1. 初始化：创建 Scene + Camera + Renderer
const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(45, w/h, 0.1, 100);
const renderer = new THREE.WebGLRenderer({ antialias: true });

// 2. 灯光：环境光 + 平行光（最常用组合）
scene.add(new THREE.AmbientLight(0xffffff, 0.5));
const dirLight = new THREE.DirectionalLight(0xffffff, 2);

// 3. 物体：创建几何体 + 材质 + 添加到场景
const mesh = new THREE.Mesh(geometry, material);
scene.add(mesh);

// 4. 交互：OrbitControls
const controls = new OrbitControls(camera, renderer.domElement);

// 5. 循环：每帧更新 + 渲染
function animate() {
  requestAnimationFrame(animate);
  controls.update();
  renderer.render(scene, camera);
}
animate();
💡 这个模板 5 步走，适用于 90% 的 Three.js 场景。以后每个项目都从这里开始。

🏋️ 扩展挑战

  1. 加阴影：给场景添加阴影（renderer.shadowMap + 物体 castShadow + 地面 receiveShadow）
  2. 加粒子：在展台周围添加浮动粒子，营造氛围感
  3. 加动画：让产品有悬浮动画（上下轻微浮动）
  4. 改布局：把产品排列成一行，做成对比展示
  5. 加背景：用渐变背景色替换纯色背景
