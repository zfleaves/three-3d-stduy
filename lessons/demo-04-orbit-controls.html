Demo 04：鼠标控制视角 — 让场景可交互

一个静止的3D场景很无聊。加上 OrbitControls（轨道控制器），用户就能用鼠标拖拽旋转、滚轮缩放、右键平移——就像在玩3D游戏一样。

🎯 本课目标：掌握 OrbitControls 的引入和配置，知道如何让用户自由探索3D场景。

🎮 试试看：拖拽、滚轮、右键

🖱️ 鼠标交互
🖱️ 左键拖拽旋转 · 滚轮缩放 · 右键平移
⚙️ 控制器设置
🔑 关键代码（只需 3 行）： 
import { OrbitControls } from 'three/addons/...';
const controls = new OrbitControls(camera, canvas);
controls.enableDamping = true; // 惯性效果

然后在渲染循环中加一行 controls.update() 即可。

📖 OrbitControls 核心操作

操作鼠标触屏效果
旋转左键拖拽单指滑动围绕目标旋转视角
缩放滚轮双指捏合拉近/拉远
平移右键拖拽双指滑动水平移动视野
🔑 常用配置：
enableDamping = true开启惯性，操作更丝滑
dampingFactor = 0.08惯性系数（越小越"粘"）
minDistance / maxDistance限制缩放范围
maxPolarAngle限制垂直旋转角度（防止翻到底部）
autoRotate = true自动旋转（展台常用）
target.set(x, y, z)设置旋转中心点
⚠️ 常见错误：忘了在渲染循环中调用 controls.update()，导致阻尼和自动旋转不生效！
